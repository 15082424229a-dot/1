# 正文

## File Convention
- `第X章.md` — X = chapter number (1, 2, 3…)
- Extras: `第X章-修订.md` for revision drafts (keep original)

## Chapter Format
```markdown
# 第X章 [optional subtitle]

[正文内容]

---
(Scene break for POV/location/time jump)
```

## Post-Writing Checklist
- [ ] `continuity-check` — 12-dimension audit vs previous chapter + character cards
- [ ] `polish-text` — surface polish pass
- [ ] `deai-slop` — de-AI pass (if needed)
- [ ] Update `大纲/伏笔台账.md` — log new hooks, mark resolved
- [ ] Update `大纲/情绪追踪.md` — note key emotional shifts
- [ ] `hook-designer` — design chapter-end hook for next chapter
