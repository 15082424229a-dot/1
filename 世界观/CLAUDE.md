# 世界观

## File Convention
| File | Content |
|------|---------|
| `原作设定.md` | Canon baseline — established world facts |
| `AU设定.md` | Alternate universe changes (if applicable) |
| `势力体系.md` | Factions, power structures, politics |
| `规则约束.md` | Hard rules — magic system, tech limits, world physics |

## Key Distinction
- **Hard settings** — cannot be violated without breaking reader trust (e.g., magic costs, established character fates)
- **Soft settings** — room for creative interpretation (e.g., minor world details, unshown periods)

## Data Flow
- → `expand-scene` (world constraints during prose expansion)
- → `continuity-check` (verify no rule violations)
