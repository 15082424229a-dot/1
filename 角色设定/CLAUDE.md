# 角色设定

## File Convention
- One card per character: `[角色名].md`
- Use the character's most common name (not titles, not nicknames)

## Required Sections
Each card must cover:
- **Basic info** — name, age, role, faction
- **Appearance** — key physical traits
- **Personality** — core traits + contradictions
- **Language style** — speech patterns, verbal tics, vocabulary range, sentence habits (critical: `dialogue-craft` depends on this)
- **Background** — history, formative events
- **Motivations** — what they want, why
- **Relationships** — links to other character cards

## Data Flow
- → `expand-scene` (char traits during prose expansion)
- → `dialogue-craft` (language style during dialogue polish)
- → `continuity-check` (compare against character behavior)
- → `emotional-arc` (personality baseline for emotion tracking)
